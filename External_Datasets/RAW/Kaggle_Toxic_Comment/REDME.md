# Kaggle Dataset

Original Paper: https://arxiv.org/pdf/1610.08914.pdf


Dataset Size:
	- Training Set: 159571
	
Class: 

A Binary 0/1 for each of these class labels (All 0's means clear class - NON-HATE)
	- toxic
	- severe_toxic
	- obscene
	- threat
	- insult
	- identity_hate
