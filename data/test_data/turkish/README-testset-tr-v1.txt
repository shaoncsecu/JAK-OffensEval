========================

OffensEval 2019: Multilingual Offensive Language Identification in Social Media (SemEval 2020 - Task 12)
Test data - Turkish (Sub-task A only)
v 1.0: Feb 2020
https://competitions.codalab.org/competitions/23020

========================

1) DESCRIPTION

The file offenseval-tr-testset-v1.tsv contains 3528 unlabeled tweets. 

You are required to upload your sub-task A predictions for each of the
3528 instances to CodaLab by no later than 26 Feb 2020 (23:59 UTC).

You will find ALL the necessary information regarding data format,
dates, number of submissions, etc. at CodaLab. 

2) FORMAT

Instances are included in TSV format as follows:

ID	INSTANCE 

The column names in the file are the following:

id	tweet

3) TASK AND LABELS

(A) Sub-task A: Offensive language identification

- (NOT) Not Offensive - This post does not contain offense or profanity.
- (OFF) Offensive - This post contains offensive language or a targeted (veiled or direct) offense

In our annotation, we label a post as offensive (OFF) if it contains
any form of non-acceptable language (profanity) or a targeted offense,
which can be veiled or direct.
